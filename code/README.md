# Cloned Repositories

## 1. Geometry of Truth
- **URL**: [https://github.com/saprmarks/geometry-of-truth](https://github.com/saprmarks/geometry-of-truth)
- **Purpose**: Official implementation of "The Geometry of Truth".
- **Location**: `code/geometry_of_truth/`
- **Relevance**: Provides methodology for finding linear truth directions and datasets for true/false statements. Useful reference for finding "humor" directions.

## 2. Eliciting Latent Sentiment
- **URL**: [https://github.com/curt-tigges/eliciting-latent-sentiment](https://github.com/curt-tigges/eliciting-latent-sentiment)
- **Purpose**: Code for "Linear Representations of Sentiment in Large Language Models".
- **Location**: `code/eliciting_latent_sentiment/`
- **Relevance**: Reference for extracting sentiment directions, which is methodologically similar to extracting humor directions.
